# dancing-man-c
Console application which shows a man dancing to the music
## Описание программы
Консольное приложение, которое демонстрирует человечка,танцующего под музыку, 
пользователь может включать и выключать мелодию нажатием одной кнопки.
## Description
A console application that demonstrates a man dancing to music,
the user can turn the melody on and off by pressing a single button.
## Featurs
- show a splash screen with pseudographics.
- view information about the author.
- viewing an opening window with a dancing man
- melody playback
## Project Structure
- main.c
## Technologies
- C
- WinAPI
- Windows Console API
- Graphical User Interface
- String Processing
- Keyboard Input Handling
- Animation Effects
## How to Run
1. Open Visual Studio
2. Build the project
3. Run the eecutable file
